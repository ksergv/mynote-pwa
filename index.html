Создание заметки

  • Введите заголовок в поле Заголовок.
  • Введите текст заметки в редакторе.
  • Нажмите Очистить для сохранения.

Редактирование/просмотр

  • Нажмите Редактировать в оласти заметки в списке, чтобы открыть её.
  • Измените текст и нажмите Сохранить.

Форматирование текста

  • B — жирный текст
  • I — курсив
  • 🖍 — подсветка текста цветом
  • HTML — режим редактирования HTML
  • 📷 — вставка картинок..
    • ... Также доступны списки, заголовки и выравнивание..

Голосовые функции

  • 🎤 — голосовой ввод текста.
  • 🔊 — озвучивание заметки.
  • 🔊 — остановить озвучивание.

Теги

  • Добавляйте теги через запятую.
  • Используйте нажатие на тег как фильтр для вывода заметок по выбранному тегу.
  • кнопка Сбросить фильтр - выводит все записи.

Советы

  • Можно вставлять текст через копирование.
  • Работает офлайн после установки приложения.
  • Заметки сохраняются только в вашем браузере.
